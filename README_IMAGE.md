# Добавление изображения Hidden Wallet

## Инструкция

Чтобы добавить изображение кошелька на главную страницу:

1. Поместите изображение в папку `public/images/` с именем `hidden-wallet.png` или `hidden-wallet.jpg`

2. Раскомментируйте код в файле `app/page.tsx` (строки с Image компонентом) и закомментируйте placeholder div

3. Убедитесь, что изображение имеет хорошее качество (рекомендуется минимум 1200x900px)

## Текущий placeholder

Сейчас на странице используется стилизованный placeholder, который имитирует дизайн кошелька из изображения. Он включает:
- Темно-синий градиентный фон
- Неоновое синее свечение
- Футуристический дизайн

## Пример использования

```tsx
<Image 
  src="/images/hidden-wallet.png" 
  alt="Hidden Wallet" 
  width={600} 
  height={450}
  className="relative z-10 drop-shadow-[0_0_30px_rgba(100,200,255,0.5)]"
/>
```
